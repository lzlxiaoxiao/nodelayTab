# nodelayTab
=======
### 无延迟切换菜单
- 一级菜单结构
- 二级菜单结构
- 切换菜单时用setTimeout做延迟
- 获取鼠标当前位置，和鼠标上一次位置与菜单上下边缘的三角形区域进行比较
- 运用向量、向量叉乘判断鼠标移动的是否在三角形内

### 最终效果
- 鼠标自然移动到子菜单
- 切换无延迟
